# 当前功能基线

本文件记录第19轮编辑器工作台 UI/UX 重构后的稳定能力、实验能力和已知问题。它用于判断后续是否继续做细节打磨或进入新的功能迭代。

## 稳定功能

- 项目 JSON 导入导出：支持 `ProjectBundle`，保留脚本、素材、角色、变量、演出字段和 prop 字段。
- Tauri 本地工程读写：桌面端支持新建、打开、保存工程，并将素材复制到工程内相对路径。
- 素材库：支持背景、角色图、BGM、音效、语音、图片和 prop 资源元数据。
- 角色库：支持角色和表情资源映射。
- 变量、条件和标签：支持项目级变量、结构化条件、节点或标签跳转和运行时逻辑日志。
- Pixi 播放器：支持背景层、角色层、PropLayer、对话层、选项层和基础屏幕效果。
- 音频播放：支持 BGM、sound、voice 三类通道，音量和静音可由播放器设置控制。
- 播放器运行时 UI：支持开始菜单、暂停菜单、存档读档、历史、设置、自动播放和最小快进。
- Web 导出包：支持 demo 和 Showcase 的静态 Web 游戏目录导出。
- 代码型动画模块：支持 `AnimationModule`、`AnimationRegistry` 和 `PlayAnimationNode`。
- PropLayer 物品系统：支持 `AssetItem type="prop"`、`ShowPropNode`、`HidePropNode`、`snapshot.props`、prop 动画模块和 Web 导出。
- Showcase Demo：`examples/showcase-game` 用于验证真实/半真实素材、动画模块、prop、音频、变量分支和 Web 导出。
- 编辑器工作台 UI：`apps/editor` 已具备顶部菜单/工具栏、左侧资源管理器、中央大场景预览、脚本指令表、右侧检查器和底部状态栏。

## 实验功能

- `animations/custom`：开发者模式 custom 动画目录，目前没有插件沙箱、签名、权限系统或安全审核链路。
- AI 动画模块模板文档：只描述未来 AI 生成动画代码时的约束，本项目当前不接 AI API。
- 粒子动画：`particle.snow` 是基础演示能力，还不是完整粒子编辑系统。
- Showcase 素材准备脚本：当前用于准备固定 Showcase 素材，不是通用素材市场或下载器。
- 桌面打包体验：Tauri 已能运行和构建，但正式安装包、自动更新和发布流程还没有产品化。

## 已知问题

- 第18轮已清理当前代码中可控的 PixiJS v8 警告来源：`Container.name` 改为 `label`，占位纹理不再把 `Graphics` 当作容器承载文本，并改用 `renderer.generateTexture({ target })`。
- 如果后续仍出现 PixiJS 警告，需要区分是项目代码残留还是第三方库内部提示。
- 第19轮已完成第一版编辑器工作台式 UI/UX 重构；后续仍可继续做视觉细节、快捷入口和可用性打磨。
- 还没有 AI 素材工坊、付费 API 接入、APIKey 管理或正式插件安全系统。
- 还没有节点图编辑器、复杂关键帧时间轴或曲线编辑器。
- Web 导出包当前生成目录，不生成 zip。

## 下一阶段建议

如果第19轮命令验收继续通过，下一步建议优先做编辑器可用性细节打磨、真实桌面打包体验或小范围回归修复；不建议在当前基线上立即继续堆新的大引擎系统。
